## Why

台灣人去日本旅遊時，常面臨語言溝通的困境：日文不夠流利、英文口說也不太行，只能用「拼裝英文 + 簡單日文單字」勉強溝通。現有語言學習 APP 不是要付費就是太死板，且沒有針對「旅遊前短期集中訓練」的產品。需要一個能在出發前快速練習旅遊實用語句的工具，目標不是學到流利，而是「夠用就好」。

## What Changes

- 建立全新的旅遊語言練習 Web APP（PWA）
- 提供 4 大旅遊場景（點餐、住宿、購物、交通），每場景 30 組對話
- 每組對話同時包含日文與英文，反映真實旅遊中日英混用的溝通策略
- 支援兩階段學習流程：跟讀基礎句型 → 模擬情境對話
- 提供五十音字卡記憶遊戲作為基礎練習
- 所有內容為固定腳本，打包為 JSON 靜態資料
- 學習進度透過 localStorage 儲存
- 手機優先的 RWD 設計，可加到主畫面當 APP 使用

## Capabilities

### New Capabilities

- `scenario-practice`: 場景式語言練習功能，包含 4 大場景 × 30 組對話的跟讀與模擬對話
- `kana-flashcard`: 五十音（平假名 + 片假名）字卡記憶遊戲
- `progress-tracking`: 學習進度追蹤與記錄（localStorage）

### Modified Capabilities

（無，全新專案）

## Impact

- 全新前端專案，無既有程式碼受影響
- 技術棧：Vue3 + Vite + Tailwind CSS
- 部署平台：Vercel
- 無後端依賴、無資料庫、無外部 API 呼叫
- 內容資料以 JSON 檔形式存放於前端專案中
